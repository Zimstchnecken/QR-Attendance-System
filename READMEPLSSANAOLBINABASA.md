
# QR Attendance Frontend (Expo)

Frontend-only React Native prototype for QR attendance. Built with Expo SDK 51 and NativeWind.

## Requirements
- Node.js (LTS recommended)
- Expo Go app on Android

## Install
```bash
npm install
```

## Run (Android)
```bash
npx expo start
```
- Scan the QR code using Expo Go
- Press `a` to open an Android emulator
